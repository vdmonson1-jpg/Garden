# Individual Plants

Per-plant tracking, journals, and harvests for every plant in the garden.

## How Plant Entries Work

Each subdirectory tracks a single plant instance from acquisition to end of life.

## Naming Convention

`{common-name}-{nn}/` where `nn` is a zero-padded two-digit number (e.g. `tomato-01`, `tomato-02`).
