# Plant Library

Deep-research reference library for every species grown in this garden.

## How to Use

Each file covers a single species or variety in depth. Files are organized by category:

- `vegetables/` — Food crops grown for their edible parts
- `herbs/` — Culinary and medicinal herbs
- `flowers/` — Ornamental and pollinator-attracting flowers
- `fruits/` — Tree and vine fruits

## Tagging Conventions

Files follow the naming pattern `{common-name}.md` (all lowercase, hyphenated).

Cross-reference individual plant instances in [plants/](../plants/) and the master [companion planting chart](companion-planting.md).
